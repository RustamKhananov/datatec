- React.js

- tailwind

## To start project locally

### Clone project
```git clone https://github.com/RustamKhananov/datatec```

### Install dependencies
```npm i```

### Run project
```npm start``` 

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
